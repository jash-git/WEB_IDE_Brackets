一般我們常見的網頁編輯器都有著相當複雜的操作介面，以及很多用得到甚至是用不到的功能，很多人會選擇純文字代碼編輯器來進行網頁開發，例如Sublime Text ，但一般的代碼編輯器並沒有針對網頁開發提供相對應的功能，以致於要預覽或修改時相當耗費時間。

今天，猿妹就和大家分享一個既好用又免費、開源且跨平台的HTML/CSS/JavaScript 前端WEB 集成開發環境（IDE工具）——Brackets。

Brackets由Adobe 以HTML、CSS、Javascript等程式語言開發，主要用於網頁設計，是一個跨平台的編輯器，支援Mac、Windows 及Linux 等作業系統。

Brackets專注於網頁開發相關用途，主要有一個程式碼高亮界面，能夠快速編輯（Quick Edit），如果在編輯HTML、CSS或Javascript遇到相關屬性，會自動跳出選項，直接選擇就能減少輸入的時間，也能減少輸入錯誤而造成的問題。

如果顏色屬性則會跳出選色器，方便你加入色彩的Hex 值或RGB值。

另外，Brackets內置的即時預覽（Live Preview）功能，當你在編寫網頁時，便能夠配合瀏覽器進行即時的更新，修改網頁程式碼後，會同步顯示於瀏覽器裡，如果配合雙屏幕使用，那在開發網頁上就更加得心應手。
